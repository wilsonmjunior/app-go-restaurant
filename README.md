## How to use 

```bash
  # Clone this repository
  git clone https://github.com/wilsonmjunior/app-go-restaurant
  
  # Access the project folder 
  cd app-go-restaurant
  
  # Install dependencies
  yarn 
  
  # Run server (json-server)
  yarn server
  
  # Run the application
  yarn android
  
```
